## 技術構成
- **プラットフォーム**: UTAGE（ページビルダー）+ Vue.js で動的レンダリング
- **フォント**: Cinzel / Shippori Mincho / EB Garamond / Noto Sans JP
- **配色**: 黒基調（#080706）＋ゴールド（#C9A84C）アクセント
- **トラッキング**: GTM × 4（GTM-PS46CR8, GTM-MH8KJJXR, GTM-N56HMFR, GTM-TSB4SG4B）

## ページ構成（上から順）

### 1. ナビゲーション（固定ヘッダー）
- ロゴ「N9 井川カップ」（Cinzel, ゴールド）
- 右側テキスト（EB Garamond, 大文字）
- 半透明背景 + ブラー効果

### 2. ヒーロー領域（100vh フルスクリーン）
- **背景**: ゴールドグラデーション放射 + 対角ゴールドラインアクセント
- **アイブロウ**: Cinzel, 大文字, ゴールド
- **イベント名**: Cinzel, ゴールド
- **メインタイトル**: Shippori Mincho, 超大型（56px〜120px）, ゴールドグラデーション
  - 2行構成（line1, line2）
- **区切り線**: 王冠アイコン + ゴールドライン
- **メタ情報**: 日時・対象者などを横並び表示（EB Garamond）
- **タグ**: 枠線付きタグ（featured タグはゴールド背景）
- **賞金ブロック**: 枠付き、ラベル + 金額（EB Garamond 40px〜64px）+ 単位
- **メインコピー**: Shippori Mincho, 太字
- **サブコピー**: Shippori Mincho, ゴールド, イタリック
- **スクロール誘導**: 下部にバーアニメーション

### 3. 概要セクション
- **グリッドレイアウト**: 2列、ゴールドボーダー区切り
- **カード構成**:
  - ラベル（Cinzel, 大文字, ゴールド）
  - メインテキスト（Shippori Mincho, 24px）
  - サブテキスト（本文）
  - ハイライトバッジ（EB Garamond, 枠付き）
- **ハイライトグリッド**: 2列、アイコン + テキスト
  - hover時: ゴールドボーダー＋背景
- **デポジットバー**: 横並び表示（ラベル + 金額 + 注記）

### 4. 井川（Ikawa）セクション
- **2列グリッド**:
  - **左**: 写真エリア（最小高さ480px）
    - 写真: `ikawa_photo.jpg`
    - 写真バッジ（名前 + 肩書き、下部オーバーレイ）
  - **右**: 情報エリア
    - タグ（ゴールド背景 + 黒文字）
    - ヘッドライン（Shippori Mincho, 22px〜32px）
    - 説明文
    - ステータスグリッド（4列: ラベル + 値）
    - 引用文（EB Garamond, イタリック, ゴールド）

### 5. 賞金構成セクション
- **セクションヘッダー**: アイブロウ + タイトル（ゴールドグラデーション）+ 装飾ライン
- **賞金テーブル**: 横並び（グリッド）
  - 各賞: 順位ラベル + 金額
  - 1位は特別スタイル（featured: ゴールドグラデーション背景、大きいフォント）
- **合計表示**: 全幅バー、ラベル + 金額
- **注記**: 小テキスト

### 6. エントリーティア（参加プラン）
- **4段階のティアカード**: 縦並びグリッド
  - RECOMMENDEDバッジ（推奨プラン、ゴールド帯）
  - デポジットラベル + 金額（EB Garamond, 28px〜36px）
  - ベネフィットリスト（→ マーカー付き）
  - プラン名（右寄せ）
  - **プレイ条件**: 区切り線下に条件行（ラベル + 値）

### 7. ベネフィットサマリー
- **2列グリッド**: アイコン + テキスト
  - featuredアイテム: 左ゴールドボーダー＋背景
  - タグ（ゴールドバッジ）
  - タイトル（Shippori Mincho）+ 説明文

### 8. スケジュール
- **タイムライン形式**: 縦並び
  - 各ステップ: 番号 + タイトル + 説明
  - ゴールド縦ライン接続
  - ステップ番号: EB Garamond, ゴールド

### 9. ルール・注釈
- **イントロ文**: カード、左ゴールドボーダー
- **ルールグリッド**: 2列
  - 各ルール: 大きな番号（EB Garamond, 48px）+ タイトル + 説明文
- **注釈セクション**: 小さな文字、リスト形式

### 10. エントリーセクション（CTA）
- **参加費表示**: EB Garamond, 大型（36px〜54px）+ 税表記
- **エントリーボタン**: ゴールドグラデーション背景, Cinzel, 大文字
  - hover時: 明るいゴールド＋シャドウ
- **LINE連携ボタン**: 緑色（#06C755）, LINEアイコン付き
- **注釈**: 小テキスト

### 11. フッター
- ロゴ（Cinzel, 28px, ゴールド）
- タグライン（EB Garamond, 大文字）
- 著作権表示

- 実際のデータ（賞金額、日時、参加費など）はスクリーンショットまたはCMS側から確認が必要
- 井川セクション写真: `ikawa_photo.jpg` として保存済み
